1. Εγκατάσταση Python απο τον ιστότοπο python.org (ο κώδικας γράφτηκε σε Python 3.9.1).
2. Εγκατάσταση Sqlite 3 απο τον ιστότοπο sqlite.org:
    2.1 Precompiled binaries sqlite-tools-<os name>-x86-3370200.zip / Για λειτουργικό Windows : και το sqlite-dll-<win32/win64>-<x86/x64>-3370200.zip
    2.2 Αποσυμπίεση σε φάκελο slite που δημιουργείται στο εξής PATH: C:/Program Files/sqlite
    2.3 (Windows) Εισαγωγή του παραπάνω path στην μεταβλητή συστήματος "PATH".
3. Εγκατάσταση των αρχείων απο το αποθετήριο στο github.
4. Άνοιγμα αρχείου db_app.py με κάποιο editor/IDE (προτείνεται το Visual Studio Code με παράλληλη εγκατάσταση του Sqlite3 extension για περιήγηση στην βάση δεδομένων).
5. Εκτέλεση αρχείου db_app.py.
